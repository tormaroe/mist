A Lisp implementation for
the .Net Framework!

“Within a couple weeks of learning Lisp I found programming
in any other language unbearably constraining.”
— Paul Graham

“Programming in Lisp is like playing with the primordial forces of the universe.
It feels like lightning between your fingertips.
No other language even feels close.”
— Glenn Ehrlich

Lisp has been called a programmable programming language, a language for doing what you've been told is impossible, and the most intelligent way to misuse a computer. Alan Key, the father of Smalltalk, said that Lisp is the greatest single programming language ever designed.

This makes it pretty obvious that there need to be Lisp implementations available for .NET developers. Mist is an attempt to provide exactly that.

Mist is currently in alpha, and the links in the menu takes you nowhere. A beta release is planned for October/November 2011. If you want to follow the progress, have a look at the github project.
